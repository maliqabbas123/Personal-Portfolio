# Abbas Ahmed Portfolio

A modern personal portfolio website for Abbas Ahmed, a Full Stack Software Engineer focused on React, Next.js, FastAPI, AI/LLM workflows, LangGraph, PostgreSQL, and Azure.

Built with:

- Next.js App Router
- TypeScript
- Tailwind CSS
- lucide-react icons
- Reusable component sections
- Editable content arrays in `src/lib/data.ts`

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

Build for production:

```bash
npm run build
```

## Editing Content

Most portfolio content is stored in:

```bash
src/lib/data.ts
```

Update skills, project cards, timeline entries, and contact links there.

Main layout sections live in `src/components/`, and the landing page is composed in `src/app/page.tsx`.
